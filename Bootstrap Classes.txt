Bootstrap Classes

2) Container-Fluid Class
3) Row Class
4) Col Class(Col-xl, Col-lg, Col-md, Col-sm )
5) W-100(use for break in rows)
6) Offset Class
7) Reordering Class
8) Nesting Class(in one col you add multiple row or col)
9) Margin Classes(ml,mr,mt,mb) Negative Margine(ml-n5, mr-n5, mt-n5, mb-n5,)
10) Padding Classes(p-5, pl-5, pr-5, pb-5, pt-5)
11) Horizontal Alignment Classes [(Entered in Row Class)Justify-Content-end, Justify-Content-Center, Justify-Content-around]
12) Vertical Alignment Classes [(Entered in Row Class) align-items-start, align-items-end, align-items-center, align-items-stretch, align-items-baseline]
13) Align Content Classes[(Entered in Row Class) align-content-start, align-content-end, align-content-center, align-content-stretch, align-content-between, align-content-around]
14) Align Self Classes[(set alignment for each col)align-self-start, align-self-end, align-self-center, align-self-stretch, align-self-baseline, align-self-auto]
15) Flex Direction Classess[flex-row-reverse, flex-column, flex-column-reverse, flex-wrape, flex-nowrape, flex-wrap-reverse,(Entered only in col class)flex-fill]
16) Display Classes[d-inline-block, d-lg-inline-block, d-none, d-flex, flex-column]
